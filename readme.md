# Tetris 3D Readme

# Installation

### Windows :

  ```cd C:\path\to\tetris```  
Where you enter the path to the .py  

  ```python app.py```

And the app should launch : there is no PIP denpedencies

### Linux

  ```cd /home/user/app/```

Where you enter the path to the .py

  ```python app.py``` or ```python3 app.py```

The app should launch too.
